RegenbogenReich baut eine einzeln adressierbare LED Kette

Die LED Kette wurde 2021 im Rahmen eines 24-tägigen Adventskalenders entwickelt. Sie basiert auf einem ESP32 Mikrokontroller und der Software WLED. Besonders ist, dass jede LED der Kette einzeln angesteuert und der LED eigene Farben zugewiesen werden kann. Damit lassen sich tolle Effekte realisieren. Zudem läßt sich das System über WLAN und einer Handy-App steuern, Alexa-Integration inklusive. :-)

Der Aufbau der Kette als auch deren Nutzung wurde als Video aufgenommen und auf YouTube veröffentlicht. Schaut auch die Reihe gerne an. Hier findet ihr den Link auf die Playliste:

Die entwickelten Bauteile, wie Platine und Gehäuse, sind als Bauplan auf GitHub veröffentlicht. Hier ist der Link zu GitHub: https://github.com/Chrischman/regenbogenreich-led-kette

Spezifikation

  • ESP32 Mikrokontroller mit WLAN-Schnittstelle
  • WLED Betriebsystem
  • LED Kette SK6812
Genutzte Software
  • FreeCAD für das Gehäusedesign
  • Fritzing für das Platinendesign
